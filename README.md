# RESTEasy Default Community Health Files

This repository contains the default community health files for the RESTEasy organization. These files provide default 
guidelines and policies for all RESTEasy repositories that don't have their own versions.

For more information about default community health files, see the [GitHub documentation](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).

## About RESTEasy

RESTEasy is a Jakarta REST (formerly JAX-RS) implementation for Java, providing a robust and feature-rich framework for
building RESTful web services. As a Commonhaus Foundation project and part of the WildFly ecosystem, RESTEasy is widely 
used in enterprise Java applications.

**Main Website**: [resteasy.dev](https://resteasy.dev)
**Main Repository**: [github.com/resteasy/resteasy](https://github.com/resteasy/resteasy)
**License**: Apache License 2.0

## Community Health Files

This repository contains the following community health files that apply to all RESTEasy repositories:

- **[Code of Conduct](CODE_OF_CONDUCT.md)** - Guidelines for community participation and expected behavior
- **[Contributing Guidelines](CONTRIBUTING.md)** - How to contribute to RESTEasy projects
- **[Support](SUPPORT.md)** - How to get help and ask questions
- **[Security Policy](SECURITY.md)** - How to report security vulnerabilities
- **[Governance](GOVERNANCE.md)** - Project governance structure and decision-making processes
- **[Code Owners](CODEOWNERS)** - Default code review assignments
- **[Funding](FUNDING.yml)** - Information about project funding and sponsorship

## Getting Involved

We welcome contributions to all RESTEasy projects! Whether you're fixing bugs, adding features, improving documentation, 
or helping with testing, your contributions are valuable to the community.

To get started:

1. Read our **[Contributing Guidelines](CONTRIBUTING.md)** for details on how to contribute
2. Review our **[Code of Conduct](CODE_OF_CONDUCT.md)** to understand our community standards
3. Check the issue tracker in the relevant repository for open issues
4. Join the discussion in GitHub Discussions or our community channels

Thank you for being part of the RESTEasy community!
